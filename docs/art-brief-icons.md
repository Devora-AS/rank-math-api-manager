# Art Brief: Rank Math API Manager Plugin Icons

## Purpose

Plugin icons for the Rank Math API Manager WordPress plugin. Used in the Plugins screen, update notifications, and "View details" modal.

## Format

- **Dimensions**: 128×128 px and 256×256 px (square)
- **Format**: PNG, 24-bit or 32-bit
- **Filename**: `icon-128x128.png`, `icon-256x256.png`
- **Location**: `assets/images/`

## Style

- **Concept**: "SEO + API bridge" — visual metaphor: connected nodes (data/API) + upward chevron (ranking/SEO)
- **Palette**: Devora Profilmanual 1.0 — **#242a56** (Hovedfarge dark), **#fcf7ff** (light bg), **#968ab6** (Sekundær), **#3432a6** (Komplementær accent). No other colors.
- **Aesthetic**: Clean, minimal, flat geometric — brand-aligned with Devora guidelines
- **Background**: #fcf7ff for proof; transparent acceptable for plugin

## Technical Notes

- PNG-first; no transparency required but acceptable
- Icons should read clearly at small sizes (e.g. 32×32 in plugin list)
- Reference for Cursor or other image generation tools
